# Privacy Policy – GeoPi (GOP)

**Last updated:** 2025-11-16  
**Developer:** Marc Wolf (19Trojan69)

GeoPi (“GOP”, “we”, “us”, “our”) is a location-based directory tool for the Pi Network ecosystem.  
Your privacy is important to us. This document explains what data is collected, how it is used, and how it is protected.

---

## 1. Information We Collect
GeoPi may process the following optional data:

- **Approximate location**, if the user voluntarily shares it with the app  
- **Search inputs** such as:
  - distance filters  
  - shop names  
  - categories  
- **Technical data** from the Pi Browser, including:
  - device type  
  - browser type  
  - basic system information

### GeoPi does *not* collect:
- personal identity data  
- Pi Wallet keys  
- phone numbers  
- email addresses  
- financial information  
- login data  

GeoPi does *not* track, store, or share any personal profile data.

---

## 2. How We Use the Data
The collected non-personal data is used exclusively to:

- show nearby Pi-accepted shops  
- calculate distances  
- sort and filter shop results  
- improve app functionality and performance  

We do **not** sell or share data with external parties.

---

## 3. Cookies
GeoPi does not use cookies or any tracking mechanisms.

---

## 4. External Services
GeoPi may display map tiles from:

- OpenStreetMap (via Leaflet / MapLibre)  
- Apple Maps (optional integration)  
- Google Maps (optional integration)

These services may process technical data according to their own Privacy Policies.

---

## 5. Data Security
- No personal data is stored by the developer.  
- All calculations happen inside the Pi Browser environment.  
- External map providers may handle basic technical requests.  

---

## 6. User Rights
Since GeoPi does not store personal data:

- Users may stop using the app at any time.  
- No data deletion request is needed, as no personal information is held.  

---

## 7. Updates to This Policy
This Privacy Policy may be updated occasionally to improve clarity or functionality.  
Changes will be reflected on this page.
